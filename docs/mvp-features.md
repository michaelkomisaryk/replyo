# MVP Features

The following features are included in the Minimum Viable Product:

## Instagram Integration
Connect the shop's Instagram account to pull in all DMs automatically. Messages sync in real time so the team always sees the latest conversations.

## Notifications
Alert the team when a new message arrives, when a client has been waiting too long for a reply, or when an order status changes.

## Dashboard
A central view showing all active chats, their priorities, and key stats — how many chats are open, how many need urgent attention, and today's order count.

## Client Cards
Each client gets a profile card showing their conversation history, order status, notes, and assigned manager. No more hunting through DMs to remember context.

## Statuses
Each chat and order can be assigned a status (e.g., New, In Progress, Waiting for Payment, Completed, Rejected) so the team always knows where things stand.

## Search
Full-text search across all messages and client cards to quickly find any conversation, order, or client by name, keyword, or date.
