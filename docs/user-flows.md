# User Flows

## Registration Flow

1. Admin visits the MicroCRM signup page
2. Enters shop name, email, and password
3. Verifies email via confirmation link
4. Connects Instagram account via OAuth
5. Lands on the dashboard — onboarding checklist shown
6. Optionally invites team members (Manager, Support Manager)

## Seller Flow

1. Admin or Manager logs in to the dashboard
2. Views the prioritized chat list
3. Opens a chat — sees client history and order status
4. Replies directly from MicroCRM (sent via Instagram)
5. Updates chat status (e.g., "Active Order", "Completed")
6. Adds notes to the client card if needed
7. Assigns chat to another team member if needed

## Chat Flow

1. Client sends a DM on Instagram
2. MicroCRM syncs the message automatically
3. Chat appears in the dashboard under "New Clients" or "Waiting for Reply"
4. Team member receives a notification
5. Team member opens the chat and replies
6. Chat status updates based on activity

## Order Flow

1. Client confirms they want to place an order
2. Manager creates an order card linked to the client's chat
3. Order status is set to "In Progress"
4. Manager updates status as the order moves through fulfillment
5. When delivered, status changes to "Completed"
6. Chat moves to "Completed Orders" priority bucket
7. After 24 hours, chat is auto-archived
