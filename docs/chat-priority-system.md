# Chat Priority System

Chats are automatically categorized into priority buckets so the team always knows what needs attention first.

## Priority Categories

### 1. New Clients
Clients who have sent their first-ever message and have not yet received a reply.

- Highest urgency — first impression matters
- Shown at the top of the dashboard
- Triggers immediate notification to the team

### 2. Waiting for Reply
Existing clients who sent a message and have not received a response within a defined time window (e.g., 1 hour).

- High urgency — client is actively waiting
- Time-since-last-message is displayed prominently
- Color-coded based on wait time (yellow → orange → red)

### 3. Active Orders
Clients who have a confirmed order currently in progress.

- Medium-high urgency
- Linked to order card with status details
- Assigned manager is highlighted

### 4. Completed Orders
Clients whose orders have been marked as delivered or closed.

- Low urgency
- Archived from main view but searchable
- Can be re-activated if client messages again

### 5. Rejected Clients
Clients who were marked as rejected (e.g., refused order, blocked, or flagged).

- Lowest priority
- Hidden from main dashboard by default
- Accessible via filter for reference
