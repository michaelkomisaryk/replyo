# MicroCRM Overview

## Project Idea

MicroCRM is a simple CRM system for small Instagram shops.  
The main goal of the system is to help businesses not lose messages from potential clients and improve communication with customers.

Many Instagram stores receive many Direct messages every day and sometimes forget to answer users or lose important chats. This creates lost sales opportunities and poor communication with customers.

MicroCRM will organize all messages in one place and help sellers manage conversations more efficiently.

---

## Problem

Small Instagram businesses often have these problems:

- Too many messages in Instagram Direct
- Missed or forgotten clients
- No simple CRM for small shops
- Difficult order tracking
- Important chats mixed with inactive chats

Because of this:
- customers wait too long for replies
- businesses lose sales
- communication becomes difficult

---

## Solution

MicroCRM will provide:

- Instagram message integration
- One dashboard for all chats
- Smart chat priority system
- Notifications for unanswered messages
- Client cards with order information
- Order status tracking
- Search and filtering system

The system will focus on simplicity and fast communication.

---

## Target Users

Main users of the system:

- Small Instagram shops
- Handmade stores
- Clothing stores
- Beauty product shops
- Local online businesses

Business size:
- 1–5 workers
- Small and medium number of daily orders

- Rejected

### Notifications
- New message alerts

### Search System
- Search clients by name or Instagram username

### Dashboard
- Active chats
- Important conversations
- Order tracking
- New clients

## Success Metrics

The project is successful if:

- businesses answer faster
- fewer messages are lost

## Risks

### Instagram API limits
Instagram may limit API access or change integration rules.

### Complex setup
Small businesses may not want difficult setup processes.

### Message synchronization issues
Messages may appear late or synchronize incorrectly.
