# Project Overview

## Project Idea

MicroCRM is a lightweight CRM system built specifically for small Instagram-based businesses. It helps shop owners manage customer conversations, track orders, and never lose a message again.

## Short Description

A simple, focused CRM that connects directly to Instagram and brings all customer chats into one organized dashboard — designed for shops with 1 to 5 workers who sell through Instagram DMs.

## Main Goal

To give small Instagram sellers a professional communication tool that eliminates missed messages, organizes client conversations by priority, and makes the sales process smooth and trackable — without the complexity of enterprise CRMs.
