# Future Features

Features planned after the MVP is live and validated:

## AI Replies
Suggest automated reply drafts based on message content and past conversations. Let the team approve and send with one click — saving time on repetitive questions.

## Telegram Integration
Support for shops that also sell via Telegram. Bring Telegram DMs into the same dashboard alongside Instagram chats.

## WhatsApp Integration
Connect WhatsApp Business accounts so shops can manage all messaging platforms in one place.

## Analytics
Dashboard with key metrics: response times, order conversion rates, busiest hours, top managers by performance, and client return rate.

## Mobile App
A native iOS and Android app so shop owners and managers can respond to clients on the go without needing a laptop.
