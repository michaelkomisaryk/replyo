# User Roles

## Roles

### Admin
The owner or head manager of the shop. Has full access to everything.

**Permissions:**
- Connect and manage Instagram integration
- Create and manage user accounts
- View all chats and orders
- Assign chats to managers
- Access analytics and reports
- Configure notification settings
- Delete or archive any record

### Manager
A senior team member responsible for handling orders and client relationships.

**Permissions:**
- View and reply to assigned chats
- Update order statuses
- Create and edit client cards
- Add notes to conversations
- View dashboard and open chats

### Support Manager
A front-line team member focused on answering incoming messages.

**Permissions:**
- View and reply to assigned chats
- Update chat status (e.g., mark as replied)
- View client card (read-only)
- Escalate chats to Manager or Admin
